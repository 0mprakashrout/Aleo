## Build Guide

The following steps are to install Aleo and Leo on your machine.

### Prerequisites

- Install `git`:
    - bit.ly/start-git
- Install `Rust`:
    - bit.ly/start-rust
- Install `VSCode`:
    - bit.ly/start-vscode

### Install Aleo and Leo

To install Aleo and Leo, run:
```
./install.sh
```
